# Лабораторна робота №6    
## Тема: Робота з колекціями в Java  

### Виконала:  
**Дубенок Діана**, група **ІМ-34**  

## Короткий опис
У роботі реалізовано власну типізовану колекцію `MyLinkedList<T>`, що відповідає варіанту:  
- **C2 = 0 → інтерфейс List**  
- **C3 = 1 → внутрішня структура — однозв’язний список**  
Тип елементів — узагальнений клас **MusicComposition** з лабораторної роботи №5.  

## Реалізований функціонал  
- створено власну реалізацію колекції через **однозв’язний список**;  
- реалізовано ключові методи інтерфейсу  
- додано **3 конструктори**:  
  1. порожній;  
  2. конструктор з одним елементом;  
  3. конструктор зі стандартної колекції;  
- у `main()` продемонстровано роботу зі списком:  
  - додавання елементів,  
  - вставка за індексом,  
  - видалення,  
  - ітерація.
